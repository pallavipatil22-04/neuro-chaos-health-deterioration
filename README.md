# Neuro-Chaos Health Deterioration Score Prediction

## About
ML model that predicts health deterioration scores from patient vital signs.

## How it works
- Used logistic map-based chaos transformation
- Trained Random Forest Regressor (R² = 0.695)
- Deployed via Streamlit for real-time risk estimation

## Tech Stack
Python, Scikit-learn, Streamlit, Pandas, Numpy

## How to Run
pip install -r requirements.txt
streamlit run app.py
